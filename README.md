# md-paint

> View github flavored markdown while editing in your favorite editor.

## Install

### CLI

```
[sudo] npm i -g md-paint
```

### Node

```
npm i --save md-paint
```

## Use

### CLI

```
md_paint path/to/readme.md
```

### Node

```js
var mdPaint = require('md-paint');

mdPaint('/path/to/readme.md');
```
